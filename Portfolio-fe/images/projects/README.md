# Project Images

Đặt ảnh thumbnail của các projects vào đây.

**Kích thước khuyến nghị:** 800x600px

**Format:** JPG, PNG

**Naming convention:** 
- project1.jpg
- project2.jpg
- ecommerce-thumbnail.jpg
- portfolio-thumbnail.jpg

## Examples:
- `ecommerce-website.jpg` - Ảnh trang E-commerce
- `portfolio-site.jpg` - Ảnh trang Portfolio
- `mobile-app.jpg` - Ảnh Mobile App
