# Skill Icons

Đặt các icon kỹ năng vào đây.

**Kích thước khuyến nghị:** 64x64px hoặc 128x128px

**Format:** SVG (preferred), PNG

**Naming convention:**
- react.svg
- csharp.svg
- nodejs.svg
- docker.svg

## Nguồn icon miễn phí:
- https://devicon.dev/ (Developer icons)
- https://simpleicons.org/ (Brand icons)
- https://iconscout.com/ (Free icons)

## Download nhanh:
```bash
# React
https://cdn.jsdelivr.net/gh/devicons/devicon/icons/react/react-original.svg

# C#
https://cdn.jsdelivr.net/gh/devicons/devicon/icons/csharp/csharp-original.svg
```
