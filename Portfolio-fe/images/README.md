# Images Folder

## Structure

### avatar/
Place your profile avatar images here.
- Recommended: Square image, 400x400px or larger
- Format: JPG, PNG, or WebP
- File naming: `avatar.jpg`, `avatar-dark.png` (for dark mode variant)

### projects/
Place your project screenshots and images here.
- Recommended: 16:9 aspect ratio (e.g., 1920x1080, 1280x720)
- Format: JPG, PNG, or WebP
- File naming: `project-name-1.jpg`, `project-name-2.jpg`

### icons/
Place skill icons and other small icons here.
- Recommended: 64x64px or 128x128px
- Format: PNG or SVG
- File naming: `skill-name-icon.png`

## Optimization Tips

1. Compress images before uploading (use TinyPNG, ImageOptim)
2. Use WebP format for better performance
3. Keep file sizes under 500KB for optimal loading
4. Use descriptive file names for better SEO
